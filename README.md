# raccoon-v3
Clean Minecraft scanner with Threading

# Packages
- requests
- threading (standard library)
- re        (standard library)
- os        (standard library)

# Features
- Clean CLI
- Fetches RANDOM servers
- Very fast with threading (just change threading range if not fast enough for you)
